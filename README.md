# The Square Repair Website

This repository contains the source code for the production website of The Square Repair, a local trade business.


## Features

*   **Case Studies:** Provides well organized case studies to showcase technical proofs of work.

*   **Functional SEO:** Embeds Schema.org JSON-LD metadata alongside canonical links and XML sitemaps to optimize search engine indexing.

*   **Secure Payment Integration:** Offers Forte CSG Checkout supported by a Cloudflare Worker proxy.
